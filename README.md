# 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on an **Indian Wedding Dataset** using Python. The analysis focuses on understanding wedding trends, expenses, guest management, event types, venue preferences, and other key insights through data cleaning, statistical analysis, and visualizations.

The project demonstrates practical data analysis techniques using the Python data science ecosystem.

---

# 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing values
- Explore wedding categories and event types
- Analyze wedding expenses
- Study guest distributions
- Identify booking and venue trends
- Perform correlation analysis
- Generate meaningful visual insights
