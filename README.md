# 💍 Indian Weddings Cost Analysis

An Exploratory Data Analysis (EDA) project focused on understanding Indian wedding trends through data cleaning, transformation, statistical analysis, and visualization using Python.

## 📌 Project Overview

This project analyzes an Indian Weddings dataset to explore spending patterns, wedding categories, locations, decorations, entertainment, and other event-related factors. The analysis transforms raw data into meaningful insights using Python and visualization techniques.

## 🎯 Objectives

* Clean and preprocess raw data
* Perform exploratory data analysis (EDA)
* Identify trends and relationships
* Generate business insights through visualizations
* Practice real-world data analysis workflows

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Analysis Performed

* Data Cleaning & Preprocessing
* Missing Value Analysis
* Feature Renaming
* Correlation Analysis
* GroupBy Operations
* Pivot Tables

* Bar Charts
* Line Charts
* Histograms
* Pie Charts
* Count Plots
